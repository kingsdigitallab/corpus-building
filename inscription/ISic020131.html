Italian translation

commentary

Abbreviation ΛΕ frequently used on Attic vases with the value of λήκυθος. However, there are no examples of this abbreviation associated with a value notation. Agostianiani suggests interpreting the alphabetical sign as a value notation (one obol), thus proposing to translate how: "One lekythos (corresponding to the value of) one obol". See also Hackl 1909, p.98.
